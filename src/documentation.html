Documentation of SMANDA Kebumen

Created by Rizal Arfiyan with adapted from Bulma

Tampilan SMANDA dibuat menggunakan teknologi preprocessor CSS yaitu SASS (SCSS) dengan mengadaptasi beberapa fitur dari framework terkenal. Dibuatnya halaman dokumentasi ini adalah bertujuan untuk memudahkan generasi selanjutnya dalam pengembangan website serta menambah wawasan akan penggunaan html dan css itu sendiri. Harapannya dengan adanya dokumentasi ini, generasi selanjutnya akan lebih berkarya dari generasi-generasi sebelumnya. Salam developer!

Penggunaan teknologi dalam sisi desain:

  1. HTML
  2. CSS (SCSS)
  3. Javascript

Software pendukung:

  1. Node JS + NPM (otomatis terinstall jika menginstall Node JS).
  2. Code Editor VS Code (rekomendasi), Sublime Text, Notepad++, dan lain lain. (pilih salah satu)

Pendukung lainnya:

  1. Icon Font Awesome 5 Free

  • Lokasi : scss/element/*

  • Isi :

    • content.scssberisi tentang syle dasar wysiwyg
    • title.scssberisi tentang pembuatan title dan subtitle
    • icon.scssbersi tentang tata letak sebuah icon

Style Guide:

  • Dengan parent .content
  • Mempunyai ukuran:
    • .small untuk ukuran kecil
    • .normal untuk ukuran normal (default)
    • .medium untuk ukuran sedang
    • .large untuk ukuran besar
